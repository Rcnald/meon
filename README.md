## How to use

| command line  | what does                          | notes               |
|---------------|------------------------------------|---------------------|
| npm install   | install dependencies(node_modules) | execute first, once |
| npm start     | run live:server,sass & tailwind    | on production       |
| npm run build | run autoprefixer                   | project is over     |
